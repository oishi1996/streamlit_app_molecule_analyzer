## アプリの実行

- 仮想環境作成</br>
    python -m venv .venv

- requirements.txt</br>
    pip install -r requirements.txt</br>
    pip freeze > requirements.txt</br>

- 仮想環境起動(win)</br>
    .venv\Scripts\activate

- app 起動</br>
    streamlit run .\streamlit_app_molecule_analyzer.py
